========================================================================================
Chem-Data-Corrector
Removes Scattering Patterns From Spectroscopy Data.
========================================================================================
**Instructions:**
- Select .txt or .IFX file to be opened
- Select new save file name and location
- Enter wavelength region where absorbance is known to be "0"
- Select if advanced Rayleigh scattering correction is needed
- Press start to begin process
